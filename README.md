Sistema de entregas mobile para gestão de pedidos de gás.

## Descrição

Aplicativo mobile para entregadores do sistema GestGAS, permitindo visualização e confirmação de entregas em tempo real.

## Tecnologias

- React Native
- Expo
- TypeScript
- React Navigation
- Axios

## Instalação

```bash
npm install
```

## Executar

```bash
npx expo start
```
