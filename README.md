## Descrição

Aplicativo mobile para entregadores do sistema GestGAS, permitindo visualização e confirmação de entregas em tempo real.

## Tecnologias

- React Native
- Expo
- TypeScript
- React Navigation
- Axios

## Instalação

```bash
npm install
```

## Execução em modo de desenvolvimento

```bash
npx expo start
```
